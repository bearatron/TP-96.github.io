How to Play
  • Press Start when you're ready — the timer won't begin until you do.
  • Click a cell, then press a number key or tap the numpad below the board.
  • Each row must contain digits 1–9 with no repeats.
  • Each column must contain digits 1–9 with no repeats.
  • Each 3×3 box must contain digits 1–9 with no repeats.
  • Orange numbers are given clues and cannot be changed.
  • Use the Pause button above the board to freeze the timer anytime.
  • You have 3 hints per game — select an empty cell then press Use Hint.
Controls
  • Arrow keys navigate between cells.
  • Number keys 1–9 fill the selected cell.
  • Backspace or Delete clears a cell.
  • Press N or click Notes Mode to toggle pencil-note mode — numbers annotate candidates without committing a value.
  • Daily Challenge resets every day with a new seeded puzzle.
  • Everyone playing on the same day gets the exact same Daily puzzle.
  • Submit your time to the leaderboard after completing the daily.
